# 💸 Expense Tracker

A Python-based Expense Tracker that helps users manage expenses, track budgets, generate reports, and export data.

## 🚀 Features

### Expense Management
- ➕ Add Expenses
- 👀 View Expenses
- ✍️ Edit Expenses
- 🔴 Delete Expenses

### Reports & Analytics
- 🔎 Search Expenses
- 🗓️ Monthly Expense Reports
- 📈 Highest Expense Finder
- 📉 Lowest Expense Finder
- 📊 Sort Expenses

### Budget Tracking
- 🎯 Set Monthly Budget
- 💰 View Budget Status
- ⚠️ Budget Exceeded Detection

### Data Management
- 📂 JSON Storage
- 📄 Export Expenses to CSV
